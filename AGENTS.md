# Agent Instructions

- Treat this repository and the local machine as private by default.
- Do not commit secrets, credentials, API keys, tokens, cookies, private URLs, local usernames, home-directory paths, hostnames, IP addresses, machine identifiers, browser/session data, local port scan output, logs, or other computer-specific details.
- Do not commit generated local environment artifacts such as `node_modules/`, `dist/`, build caches, logs, or scan snapshots.
- Before staging or committing, inspect the diff for sensitive or machine-specific data.
- Prefer examples and placeholders over real local values in documentation and committed fixtures.
